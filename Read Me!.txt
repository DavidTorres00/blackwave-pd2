-- Installation:
Place the BLACKWAVE folder inside your PAYDAY 2 mods directory:
    .../Steam/steamapps/common/PAYDAY 2/mods/BLACKWAVE/
Requires SuperBLT. Customize Trainer/config.lua to your taste.

-- F.A.Q. :

Q? My DLC items locked again / DLC unlocker not working?
A! Set DLCUnlocker = true in config.lua. Disabled by default to avoid cheater tags.

Q? Achievements not working?
A! Stats sync disabled by default. Set NoStatsSynced = false in config.lua.
   Also set your Steam profile visibility to non-public.

Q? Game crashes on main menu after installing?
A! BLACKWAVE includes a BeardLib compatibility fix. If crash persists,
   check mods/logs/YYYY_MM_DD_log.txt and report the error.

Q? Nothing shows up in game?
A! Verify folder is named exactly BLACKWAVE inside the mods directory.
   Confirm SuperBLT is installed and working.

Q? How do I enable an option?
A! Check config.lua and explore the in-game menus.

Q? Can I be banned for using this?
A! PAYDAY 2 has no server-side ban system. You can receive a visual CHEATER
   tag from other players if you use certain client-side exploits.
   See detection notes below.

-- Detection notes:
    From client side:
     - Spawning equipment different from last spawned.
     - Throwing more grenades than allowed.
     - Spawning more equipment than allowed.
     - Securing/spawning bag different from what you carry.
     - Wearing DLC item you don't own.

    From server side:
     - Same as above, plus hosting heist from unowned DLC.
